# Gude-Mobile
## Folder Structure
```
frontend/
├── main.dart                          # Entry point + providers
├── core/                              # App-wide stuff
│   ├── constants/                     # colors, strings (ZAR, NSFAS), routes
│   ├── theme/                         # app_theme.dart (red/white + dark mode for students)
│   ├── utils/                         # extensions, offline helper, currency formatter
│   └── di/                            # dependency injection (get_it or riverpod)
├── features/                          # Each big feature in its own folder
│   ├── auth/
│   │   ├── data/                      # models + firebase datasource
│   │   ├── domain/                    # use cases (verify_student_status)
│   │   └── presentation/
│   │       ├── screens/               # splash, signup, login, forgot, student_verify, banking_capture
│   │       └── widgets/
│   ├── marketplace/
│   │   ├── data/
│   │   ├── domain/
│   │   └── presentation/
│   │       ├── screens/               # marketplace_home, listing_detail, create_listing, job_dashboard
│   │       ├── widgets/               # listing_card, category_chip, nearby_feed
│   │       └── controllers/           # riverpod providers
│   ├── wallet/
│   │   ├── data/
│   │   ├── domain/
│   │   └── presentation/
│   │       ├── screens/               # wallet_home (balance + graph), transaction_history, budget_planner, savings_goals
│   │       └── widgets/               # balance_card, spending_alert, progress_bars
│   ├── stability/
│   │   ├── data/
│   │   ├── domain/                    # AI Risk Engine logic (local first)
│   │   └── presentation/
│   │       ├── screens/               # health_dashboard (green/yellow/red), weekly_checkin, support_hub
│   │       └── widgets/
│   └── shared/                        # Reusable across features
│       ├── models/                    # Student, Listing, Transaction, StabilityScore
│       ├── widgets/                   # bottom_nav_bar, offline_banner, student_profile_card
│       └── services/                  # firebase_service, connectivity_service
├── navigation/
│   └── app_router.dart                # GoRouter (deep links + bottom nav)
└── generated/                         # (auto-generated by build_runner)
```
